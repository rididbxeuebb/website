I am rididbxeuebb, a software engineer surviving entirely on black coffee, irrational grudges against bloated tech stacks, and an unapologetic obsession with Neovim.

This site is my dumping ground. It’s where I write about the absolute state of modern software development, a horrifying landscape where a "simple" web app requires a 4GB `node_modules` folder, 37 build steps, and an AI agent hallucinating half your logic. I write down what I'm thinking because if I don't, I will scream it at my monitor and my neighbors will call the police again.

Expect unhinged rants about JavaScript atrocities, obsessive deep-dives into terminal tools, aggressive defense of open source before it got corporatized, and the occasional psychotic breakdown over why everyone refuses to just write code that actually works.

If you're here for sanitized "thought leadership" or LinkedIn-ready tutorials on how to empower your synergies, close the tab right now before you hurt yourself. If you're here because you're also tired of the bullshit, grab a seat. We're all mad down here.
